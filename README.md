# 🌐 Pradeep Kumar — Portfolio 2026

A personal portfolio website built with **React + Vite + Tailwind CSS**, showcasing my projects, skills, education, and what I'm currently exploring as a Data Scientist & Web Developer.

---

## 🚀 Live Demo

> [https://prdip01.github.io/portfolio2026](https://prdip01.github.io/portfolio2026) *(deploy link — update when live)*

---

## ✨ Features

- ⚡ Built with **React + Vite** for fast performance
- 🎨 Styled with **Tailwind CSS** — dark brutalist / editorial aesthetic
- 🎞️ Smooth animations via **Framer Motion**
- 📱 Fully **responsive** — mobile, tablet, desktop
- 🧠 Sections: Hero, About, Projects, Skills, Education, Currently Exploring, Contact
- 🔗 Direct links to GitHub, Instagram, and email

## 🧑‍💻 Getting Started

### Prerequisites

- Node.js ≥ 18
- npm ≥ 9

### Installation

```bash
# Clone the repo
git clone https://github.com/prdip01/portfolio2026.git
cd portfolio2026

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
